imgsquare
=========

VRChatのワールド「The Avatar Studio | Avatown」でThumbnail撮影した画像の
アバターの正方形部分だけ切り出すだけの自分用のツールです。


================================================================
基本的な使い方
================================================================

imgsquare.exe に対象の画像をドラッグ＆ドロップするだけです。

エクスプローラーの「送る」メニューから使いたい場合は、imgsquare.exe の
ショートカットを以下のフォルダに置いておくと便利です。

  %APPDATA%\Microsoft\Windows\SendTo

デフォルトでは以下の動作になります。

  - 切り出し位置 : 左端から（start）
  - 出力サイズ   : 1080 x 1080 px
  - 出力先       : 元画像と同じフォルダ
  - 出力ファイル名: 元ファイル名_サイズ.拡張子

同名ファイルが既に存在する場合はファイル名に連番が付きます。

  例）
    avatar.png            → avatar_1080.png
    avatar_1080.png が既にある  → avatar_1080 (1).png
    avatar_1080 (1).png が既にある → avatar_1080 (2).png


================================================================
設定ファイル (imgsquare.conf)
================================================================

初回起動時に imgsquare.exe と同じフォルダに imgsquare.conf が
自動生成されます。よく使う設定をここに書いておくと楽です。

  {
    "imagepath": null,
    "outdir": "",
    "outimage": "",
    "position": "Start",
    "size": 1080,
    "expand": false
  }


================================================================
コマンドラインオプション
================================================================

Usage: imgsquare [OPTIONS] [IMAGEPATH]

引数:
  [IMAGEPATH]              入力画像ファイルのパス

オプション:
  -d, --outdir <OUTDIR>    出力先フォルダのパス
                           省略時は入力画像と同じフォルダに出力

  -o, --outimage <OUTIMAGE>
                           出力ファイル名
                           省略時は「元ファイル名_サイズ.拡張子」

  -p, --position <POSITION>
                           切り出し位置（start / center / end）
                           省略時は start

  -s, --size <SIZE>        出力する正方形のサイズ（ピクセル）
                           省略時は 1080

  -e, --expand [true|false]
                           入力画像の短辺が size 未満のとき拡大するか
                           -e のみでも true 扱いになります
                           省略時は false（拡大しない）

  -c, --config <CONFIG>    設定ファイルのパス
                           省略時は imgsquare.exe と同フォルダの
                           imgsquare.conf を使用

  -h, --help               ヘルプを表示
  -V, --version            バージョンを表示


-- position について --

切り出し位置は画像の「長辺方向」で決まります。

  縦長画像（高さ > 幅）の場合:
    start  → 上から
    center → 中央
    end    → 下から

  横長画像（幅 > 高さ）の場合:
    start  → 左から
    center → 中央
    end    → 右から
